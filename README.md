# MLPR Lab 5 – Distance-Based Classification

# Aim
To implement face detection using Haar Cascade and perform distance-based classification using HSV features and K-Means clustering.

# Methodology
1. Face detection using Haar cascade from the OpenCV python library
2. conversion of the images from BGR to HSV color space
3. Extraction of the features Hue and saturation from the image
4. Application of K-Means clustering (using k=2) for classification of all the faces
5. Classification of a template image based on predicted cluster
